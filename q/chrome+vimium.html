Remember

  • F6：切换焦点到不同窗格（比如可以切换到地址栏，书签栏）
  • Ctrl + L： 选中当前标签页的URL地址，好直接输入新网址

    搜索组合键

  • Ctrl + F / F3：搜索
  • Ctrl + G：找到通过Ctrl+F查找的下一个词
  • Ctrl + Shift + G：则是查找上一个词

Common

  • Ctrl + W：关闭网页（vim替换为x）
  • Ctrl + R / F5：刷新网页（按住几秒可以强制刷新dns）（vim替换为r）
  • Ctrl + Tab：切换到下一个标签页（vim替换为Shift + K）
  • Ctrl + Shift + Tab：切换到上一个标签页（vim替换为Shift + J）
  • Ctrl + N：打开一个新窗口
  • Ctrl + Shift + N：打开一个新隐身窗口

Know

  • Ctrl +J：打开下载页
  • Ctrl + Shift + B：show or hide bookmark
  • Alt + Home：打开主页
  • Ctrl + 1(2,3,4等等)：切换到第几个标签页
